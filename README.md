# Sales Order Application
React + Tailwind + .NET Core + SQL Server

## Features
- Create Sales Orders
- Edit Sales Orders
- Auto-calc totals (Excl, Tax, Incl)
- Client & Item lookup
- Double-click order to edit
- Fully connected React frontend + .NET API

## Tech Stack
- .NET Core Web API
- Entity Framework Core
- SQL Server
- React (Vite)
- Tailwind CSS
- Axios

## How to Run (Backend)
cd src/SalesOrderApp.API
dotnet run

## How to Run (Frontend)
cd frontend
npm install
npm run dev

## API Base URL
http://localhost:5059/api
